Un test de conformité est un test statistique permettant de vérifier si un échantillon de données est compatible avec une distribution théorique ou une hypothèse donnée. Le test du khi-deux d'ajustement est le test de conformité le plus classique : il compare les fréquences observées aux fréquences théoriques prévues sous l'hypothèse nulle. D'autres tests de conformité incluent le test de Kolmogorov-Smirnov et le test de Shapiro-Wilk pour la normalité. Ces tests sont essentiels pour valider les hypothèses statistiques sur les distributions.
